app.coffee

¶

Root level operations for all the unit tests.

"requirejs", "request" (supertest), "async", and "app" are initialized here and can referenced in all unit tests as long as they are nested within:

describe "app", ->
  it "should not be null" () =>
    @app.should.not.equal null
